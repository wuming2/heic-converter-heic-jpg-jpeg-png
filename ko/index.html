heic 변환 / HEIC to JPG

빠르고 안전한 브라우저 기반 heic 변환 도구

이 heic 변환 도구는 iPhone 사진, 앨범 이미지, 업무용 이미지를 빠르게 처리해야 하는 사용자를 위한 도구입니다. HEIC 또는 HEIF 파일을 JPG, JPEG, PNG처럼 널리 쓰이는 형식으로 변환할 수 있습니다.

일괄 처리

여러 HEIC/HEIF 이미지를 한 번에 추가하고 변환 상태와 출력 크기를 확인할 수 있습니다.

로컬 변환

파일을 서버에 업로드하지 않고 브라우저에서 직접 디코딩, 인코딩, 다운로드합니다.

출력 제어

기본 PNG 출력은 디코딩된 픽셀 품질을 보존하며, JPEG/JPG 품질도 조절할 수 있습니다.

EXIF 옵션

EXIF 삭제를 켤 수 있으며, 끄면 인식 가능한 원본 EXIF 정보를 최대한 보존합니다.

JPG, JPEG, PNG 중 무엇을 선택해야 하나요?

웹사이트 업로드, 문서 첨부, Windows 또는 Android 기기와의 공유에는 JPG/JPEG가 가장 무난합니다. 픽셀 품질이나 이후 편집이 더 중요하다면 PNG가 더 적합합니다.

EXIF 삭제가 필요한 이유

EXIF에는 촬영 기기, 시간, 방향, 렌즈 정보, 경우에 따라 위치 관련 데이터가 포함될 수 있습니다. 공개 공유 전에는 EXIF 삭제를 켜는 것이 좋습니다.

자주 묻는 질문

HEIC 파일이 서버로 업로드되나요?

아니요. 현재 버전은 순수 웹 프런트엔드로 동작하며 파일은 브라우저 안에서 처리됩니다.

기본 무손실 변환은 어떤 의미인가요?

기본 PNG 출력은 디코딩된 픽셀을 손실 압축 없이 저장합니다. JPG/JPEG는 다시 인코딩되므로 엄밀히 말해 무손실은 아닙니다.

저장 폴더 선택이 일부 브라우저에서만 동작하는 이유는?

폴더 선택은 File System Access API에 의존합니다. 지원하지 않는 브라우저에서는 ZIP 다운로드로 자동 전환됩니다.
