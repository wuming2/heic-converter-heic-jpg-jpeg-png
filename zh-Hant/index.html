EXIF 選項

可清除 EXIF；關閉時會盡量保留可識別資訊。

什麼時候選 JPG、JPEG 或 PNG？

表單、郵件與 Windows/Android 相容性通常選 JPG/JPEG；重視品質或後續編輯則選 PNG。

為什麼要清除 EXIF？

EXIF 可能包含設備、時間、方向、鏡頭甚至位置資料。公開分享前建議清除。

常見問題

HEIC 檔案會上傳到伺服器嗎？

不會。此工具在瀏覽器本地處理檔案。

HEIC 轉 JPG 是無損嗎？

JPG/JPEG 需要重新編碼，不是嚴格無損；PNG 會無損保存解碼後像素。

為什麼資料夾儲存只支援部分瀏覽器？

它依賴 File System Access API；不支援時會改用 ZIP 下載。
