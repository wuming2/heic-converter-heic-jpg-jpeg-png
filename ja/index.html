HEIC to JPG / JPEG / PNG

ブラウザで高速・安全に HEIC 画像を変換

iPhone の HEIC/HEIF 写真を JPG、JPEG、PNG に変換できます。処理はブラウザ内で行われます。

一括処理

複数の HEIC/HEIF 画像を追加し、状態や出力サイズを確認できます。

ローカル変換

サーバーへアップロードせず、ブラウザで直接変換します。

出力制御

PNG は無損失、JPEG/JPG は品質調整に対応します。

EXIF オプション

共有前に EXIF を削除、または必要に応じて保持できます。

JPG、JPEG、PNG の選び方

アップロードや Windows/Android 共有には JPG/JPEG、画質や編集を重視するなら PNG が適しています。

EXIF 削除が必要な理由

EXIF には撮影機器、日時、向き、場合によって位置情報が含まれます。公開前の削除がおすすめです。

よくある質問

HEIC ファイルはサーバーへアップロードされますか？

いいえ。ファイルはブラウザ内で処理されます。

HEIC から JPG は無損失ですか？

JPG/JPEG は再エンコードされるため厳密には無損失ではありません。PNG はデコード後のピクセルを無損失で保存します。

保存先フォルダ選択が一部ブラウザだけなのはなぜ？

File System Access API に依存します。未対応ブラウザでは ZIP ダウンロードを使用します。
