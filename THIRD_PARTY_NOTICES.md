# Third-party Notices

This static app vendors browser builds of these open-source libraries:

- `heic2any` for browser HEIC/HEIF decoding and conversion.
- `heic-to` for HEIC/HEIF decoding with newer libheif builds.
- `piexifjs` for inserting EXIF into JPEG output.
- `JSZip` for fallback batch downloads. The vendored browser build states it is dual licensed under MIT or GPLv3.
- `lucide` for UI icons. The vendored browser build states it is licensed under ISC.

Keep each vendored file's license header intact when updating dependencies.
