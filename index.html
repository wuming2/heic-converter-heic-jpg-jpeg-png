什么时候选择 JPG、JPEG 或 PNG？

如果你要把 iPhone HEIC 照片发给网站、表单、旧版软件或 Windows 设备，JPG/JPEG 通常兼容性最好。 如果你更重视像素质量、截图边缘或后续编辑，PNG 更合适。由于 JPEG 是有损格式，严格意义上的无损转换建议使用 PNG。

为什么加入 EXIF 清洗？

EXIF 可能包含拍摄设备、时间、方向、镜头信息，部分照片还可能带有位置相关数据。 发布到公开网页、简历、工单或客户资料前，可以打开 EXIF 清洗，减少不必要的信息暴露。

常见问题

HEIC 文件会上传到服务器吗？

不会。当前实现是纯 Web 前端，文件在你的浏览器内处理。

默认“无损转换”是什么意思？

默认 PNG 输出不会对解码后的像素做有损压缩。HEIC 到 JPG/JPEG 则需要重新编码，因此不是严格无损。

为什么保存路径只在部分浏览器可用？

目录选择依赖 File System Access API，主要由 Chromium 系浏览器支持；其他浏览器会自动使用 ZIP 下载。
