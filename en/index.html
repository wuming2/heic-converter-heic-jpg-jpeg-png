When should I choose JPG, JPEG, or PNG?

JPG/JPEG is best for uploads, forms, email, and Windows or Android compatibility. PNG is better when pixel quality or later editing matters.

Why clean EXIF?

EXIF can contain device, date, orientation, lens, and sometimes location data. Cleaning it helps before public sharing.

FAQ

Are HEIC files uploaded to a server?

No. This static app processes files locally in your browser.

Is HEIC to JPG lossless?

JPG/JPEG is re-encoded, so it is not strictly lossless. The default PNG output preserves decoded pixels losslessly.

Why does folder saving work only in some browsers?

Folder selection relies on the File System Access API. Unsupported browsers use ZIP download instead.
